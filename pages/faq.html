Questions, answered

Everything you need to know before you plug the agent into your stack.

What powers the agent?

A FastAPI core orchestrates DuckDB, async workers, and OpenAI-backed reasoning. The UI ships as a Svelte + Vite experience mirrored here as a static preview.

Can I deploy this quickly?

Yes. Drop the static bundle into GitHub Pages or any CDN-backed host. For the full application, follow the Docker or UV workflows in the primary repository.

How do updates roll out?

Sync from the main repository to ensure UI changes, worker improvements, and schema migrations land together. A changelog is published with every release.

Is the agent secure?

OAuth-based authentication, scoped API secrets, and encrypted DuckDB configuration keep access contained. No secrets are stored in this static preview.

Where can I follow progress?

Watch the public roadmap inside the main workspace. Release notes, demos, and quickstart guides live there as well.
